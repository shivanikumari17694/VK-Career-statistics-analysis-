# VK-Career-statistics-analysis-
This project presents an interactive Power BI dashboard analysing the career performance of Virat Kohli across different formats and years

🚀 Project Overview
The dashboard provides a detailed analysis of Virat Kohli's cricket career using key performance metrics and visualisations. It helps users understand trends, consistency, and achievements over time.

📌 Key Features
📅 Year-wise Analysis – Track performance trends over different years
🏏 Total Runs & Innings – Overview of career achievements
📊 Batting Average & Strike Rate – Performance efficiency metrics
🔢 Boundaries Analysis – Distribution of 4s and 6s
💯 Top Scores – Number of 50s and 100s
🎯 Filters/Slicers – Analyse data by year and format (Test, ODI, T20)

🛠️ Tools & Technologies Used
Power BI – Data visualisation and dashboard creation
Excel / CSV – Data source
DAX (Data Analysis Expressions) – Calculations and measures
